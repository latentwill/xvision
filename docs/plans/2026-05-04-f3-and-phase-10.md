# F3 (Trader-as-Strategy) + Phase 10 (Demo polish) — Implementation Plan

> **2026-05-07: Historical record per ADR 0011.** F3's `VectorConfig`
> four-arm framing has been collapsed (CV substrate moved to xianvec-play);
> TraderArm now ships as a single LLM-driven Strategy variant. Phase 10's
> `xvn explain-vectors` subcommand has been removed. The rest of F3
> (Strategy adapter + async lift) and Phase 10 (Markdown report generator)
> survives unchanged.

> **Date:** 2026-05-04 · **Branch:** `phase-0-1` · **Scope:** GPU-independent
> work that sets up Phase 9's A/B plug and ships the venue-agnostic demo
> surface. F1, F2, F4–F22 stay deferred per FOLLOWUPS.md.

**Goal:** Ship the `Strategy`-shaped wrapper around the Intern + Trader pipeline
(F3) and the four `xvn` demo commands + Markdown report generator (Phase 10.1
+ 10.2), with all four `VectorConfig` arms wired but only `Off` runtime-tested
until F1/F2 land.

**Architecture:**
- The `Strategy` trait becomes `async` (via `#[async_trait]`) so an LLM-backed
  strategy can `.await` the Intern HTTP call inside `decide`. All seven existing
  baselines convert to `async fn decide` (no awaits inside — purely mechanical).
- A new `TraderArm` adapter in `crates/xianvec-eval/src/baselines/trader_arm.rs`
  composes `BriefingCache` → `InternBackend::brief` → `run_trader` → `TraderDecision`,
  parameterised by `VectorConfig { Off | On | Random | Orthogonal }`.
- `xianvec-eval` gains transitive deps on `xianvec-trader`, `xianvec-intern`,
  `xianvec-inference`. Heavy, but it's where `Strategy` lives.
- `xvn` grows four subcommands plus a `report` command. Phase 10.1's `run-setup`
  is wired against the same intern/trader/risk surface as `TraderArm`; the
  `ab-compare` runner stays Phase 9's job.

**Tech Stack:** Rust 1.95 · `async-trait` (already a workspace dep) · `clap` ·
existing `xianvec-eval` metrics/bootstrap/gate · `serde_json` for reports.

```rust
//! TraderArm — the Strategy adapter that runs a real Intern + Trader
//! pipeline against a `MarketSnapshot`. The four `VectorConfig` variants
//! cover Phase 9.2's experimental arms (off, on, random, orthogonal).
//!
//! v1 only `Off` is end-to-end tested in CI. The `On / Random / Orthogonal`
//! variants compile + load but their directional-match validity rides on
//! F1 (spike directional match through candle) + F2 (production vector
//! extraction).
//!
//! ## Briefing pairing (Tier 1 fix #1)
//! All arms share a `BriefingCache`; the cache key is
//! `(setup_id, intern_provider, intern_model)`. Vector arms differ only in
//! Stage 2 (which vector is loaded into Qwen3Engine), never in Stage 1.

use std::path::PathBuf;
use std::sync::Arc;

use async_trait::async_trait;
use tokio::sync::Mutex;

use xianvec_core::trading::{DispositionAxis, PortfolioState, TraderDecision};
use xianvec_core::market::MarketSnapshot;
use xianvec_core::Manifest;
use xianvec_inference::engine::Qwen3Engine;
use xianvec_intern::{BriefingCache, cache::CacheKey, InternBackend};
use xianvec_trader::TraderParams;

use crate::strategy::Strategy;

/// Which vector (if any) the arm loads into Qwen3Engine before running the
/// Trader. `Off` is the no-vector control; the other three are experimental
/// arms whose validity rides on F1 / F2.
#[derive(Debug, Clone)]
pub enum VectorConfig {
    Off,
    On {
        manifest: Manifest,
        npz_path: PathBuf,
        alpha: f32,
    },
    Random {
        seed: u64,
        layer: u16,
        hidden_dim: usize,
        alpha: f32,
    },
    Orthogonal {
        axis: DispositionAxis,
        seed: u64,
        npz_path: PathBuf,
        alpha: f32,
    },
}

impl VectorConfig {
    pub fn label(&self) -> &'static str {
        match self {
            VectorConfig::Off => "off",
            VectorConfig::On { .. } => "on",
            VectorConfig::Random { .. } => "random",
            VectorConfig::Orthogonal { .. } => "orthogonal",
        }
    }
}

/// The Strategy adapter.
pub struct TraderArm {
    arm_name: &'static str,
    intern: Arc<dyn InternBackend>,
    intern_provider: String,
    intern_model: String,
    cache: Arc<BriefingCache>,
    /// Trader engine guarded by a Mutex — Qwen3Engine is `&mut self` for
    /// generation. Sequential decoding per arm is fine because the harness
    /// already serialises arms inside its decision loop.
    engine: Arc<Mutex<Qwen3Engine>>,
    trader_params: TraderParams,
    vector_config: VectorConfig,
    /// Snapshot owner of portfolio state (the harness owns the executor's
    /// portfolio; v1 hands the adapter a stub portfolio because the trader
    /// uses portfolio state only as a sanity prompt input — the executor
    /// then enforces sizing). The harness will wire the real portfolio in a
    /// later refactor; for v1 we accept a closure provider.
    portfolio_provider: Arc<dyn Fn() -> PortfolioState + Send + Sync>,
}

impl TraderArm {
    pub fn new(
        arm_name: &'static str,
        intern: Arc<dyn InternBackend>,
        intern_provider: impl Into<String>,
        intern_model: impl Into<String>,
        cache: Arc<BriefingCache>,
        engine: Arc<Mutex<Qwen3Engine>>,
        trader_params: TraderParams,
        vector_config: VectorConfig,
        portfolio_provider: Arc<dyn Fn() -> PortfolioState + Send + Sync>,
    ) -> Self {
        Self {
            arm_name,
            intern,
            intern_provider: intern_provider.into(),
            intern_model: intern_model.into(),
            cache,
            engine,
            trader_params,
            vector_config,
            portfolio_provider,
        }
    }
}

#[async_trait]
impl Strategy for TraderArm {
    fn name(&self) -> &'static str {
        self.arm_name
    }

    async fn decide(&self, snapshot: &MarketSnapshot) -> Option<TraderDecision> {
        // Implemented in Step 5.x.
        let _ = (&self.intern, &self.cache, &self.engine, &self.trader_params,
                 &self.vector_config, &self.portfolio_provider, snapshot);
        None
    }
}

#[cfg(test)]
mod tests {

            VectorConfig::Off.label(),
            VectorConfig::Random { seed: 0, layer: 20, hidden_dim: 5120, alpha: 1.0 }.label(),
        ];
        assert_eq!(labels[0], "off");
        assert_eq!(labels[1], "random");
    }
}
```

- [ ] **Step 4.2: Implement `TraderArm::decide` for the Off arm**

```rust
async fn decide(&self, snapshot: &MarketSnapshot) -> Option<TraderDecision> {
    use xianvec_intern::prompt::{build_intern_prompt, PromptOpts};
    use xianvec_trader::run::{run_trader};

    // 1. Stage 1: cached or fresh briefing
    let key = CacheKey {
        setup_id: snapshot.setup_id,
        provider: self.intern_provider.clone(),
        model: self.intern_model.clone(),
    };
    let briefing = if let Some(b) = self.cache.get(&key) {
        b
    } else {
        let prompt = build_intern_prompt(snapshot, &PromptOpts::default());
        match self
            .intern
            .brief(
                &prompt,
                snapshot.setup_id,
                snapshot.asset,
                snapshot.regime,
                snapshot.horizon_hours,
            )
            .await
        {
            Ok(b) => {
                self.cache.insert(key, b.clone());
                b
            }
            Err(e) => {
                tracing::warn!(target: "trader_arm", arm = self.arm_name,
                               error = %e, "intern brief failed");
                return None;
            }
        }
    };

    // 2. Vector load (v1: Off only). On/Random/Orthogonal short-circuit
    //    with a warn until F1/F2 land — they remain compile-checked but
    //    unused in CI.
    if !matches!(self.vector_config, VectorConfig::Off) {
        tracing::warn!(target: "trader_arm", arm = self.arm_name,
                       config = self.vector_config.label(),
                       "non-Off VectorConfig requested; F1/F2 not yet landed — \
                        running the Trader without an installed vector. \
                        This arm's directional claim is invalid until F2.");
        // Intentionally fall through to vectors-off behaviour rather than
        // refusing — the harness shape exercises end-to-end.
    }

    // 3. Stage 2: trader call (Mutex-guarded engine).
    let portfolio = (self.portfolio_provider)();
    let mut engine = self.engine.lock().await;
    match run_trader(&mut *engine, &briefing, &portfolio, &self.trader_params) {
        Ok(d) => Some(d),
        Err(e) => {
            tracing::warn!(target: "trader_arm", arm = self.arm_name,
                           error = %e, "trader failed; emitting None");
            None
        }
    }
}
```

```rust
pub mod commands;

use clap::{Parser, Subcommand};
use std::path::PathBuf;
use uuid::Uuid;

#[derive(Parser, Debug)]
#[command(name = "xvn", version, about = "XIANVEC: vectors-on vs vectors-off trading agent")]
pub struct Cli {
    #[command(subcommand)]
    pub command: Command,
}

#[derive(Subcommand, Debug)]
pub enum Command {
    /// Render a backtest report's headline Δ-Sharpe + dashboard.
    ShowMetrics {
        #[arg(long)]
        report: PathBuf,
    },
    /// Pretty-print a cached TraderDecision by setup_id (SQLite store).
    ShowDecision {
        #[arg(long)]
        setup_id: Uuid,
        #[arg(long, default_value = "data/store.db")]
        db: PathBuf,
    },
    /// Print the active disposition axis manifest + spike validation summary.
    ExplainVectors {
        #[arg(long)]
        manifest: PathBuf,
    },
    /// Run a single setup end-to-end (Intern → Trader → Risk).
    RunSetup {
        #[arg(long)]
        setup_id: Uuid,
        #[arg(long, default_value = "config/default.toml")]
        config: PathBuf,
    },
    /// Render the headline Markdown report for a backtest run.
    Report {
        #[arg(long)]
        input: PathBuf,
        #[arg(long)]
        output: PathBuf,
    },
}

impl Cli {
    pub async fn run(self) -> anyhow::Result<()> {
        match self.command {
            Command::ShowMetrics { report } => commands::show_metrics::run(report),
            Command::ShowDecision { setup_id, db } => commands::show_decision::run(setup_id, db).await,
            Command::ExplainVectors { manifest } => commands::explain_vectors::run(manifest),
            Command::RunSetup { setup_id, config } => commands::run_setup::run(setup_id, config).await,
            Command::Report { input, output } => commands::report::run(input, output),
        }
    }
}
```

```rust
//! Phase 10.2 — Markdown report generator.
//!
//! Reads a `BacktestResult`, computes Δ-Sharpe + bootstrap CI for each pair
//! of arms (relative to a designated baseline arm), runs the anti-overfit
//! gate by-regime, and renders Markdown with:
//! 1. Headline Δ-Sharpe (treatment − baseline) + 95% CI
//! 2. Secondary metrics dashboard (per arm: Sharpe, MDD, PF, WR, total PnL)
//! 3. Regime-stratified Δ-Sharpe with the gate verdict
//! 4. Decision-divergence table (action+direction+size_bucket)
//!
//! Inferential vs descriptive distinction is stated explicitly (Tier 3
//! review item from implementation-plan.md §90).

use std::collections::BTreeMap;
use std::fmt::Write;

use crate::bootstrap::{bootstrap_delta_sharpe, BootstrapConfig};
use crate::gate::{anti_overfit_gate, GateVerdict};
use crate::metrics::{drawdown_max, profit_factor, sharpe_annualized, win_rate, PreCommittedMetrics};
use crate::result::BacktestResult;

#[derive(Debug, Clone)]
pub struct ReportConfig {
    pub baseline_arm: String,
    pub n_bootstrap_resamples: usize,
    pub block_size: Option<usize>,
    pub periods_per_year: f32,
    pub seed: u64,
}

impl Default for ReportConfig {
    fn default() -> Self {
        Self {
            baseline_arm: "vectors_off".into(),
            n_bootstrap_resamples: 1000,
            block_size: None,
            periods_per_year: 252.0,
            seed: 42,
        }
    }
}

pub fn render(result: &BacktestResult, cfg: &ReportConfig) -> anyhow::Result<String> {
    let mut s = String::with_capacity(4096);
    writeln!(s, "# XIANVEC backtest report")?;
    writeln!(s)?;
    writeln!(s, "- setups evaluated: {}", result.setups_evaluated)?;
    writeln!(s, "- initial NAV: ${:.2}", result.initial_nav_usd)?;
    writeln!(s, "- started: {}", result.started_at)?;
    writeln!(s, "- finished: {}", result.finished_at)?;
    writeln!(s, "- baseline arm: `{}`", cfg.baseline_arm)?;
    writeln!(s)?;

    // -- Headline ------------------------------------------------------
    writeln!(s, "## Headline Δ-Sharpe (95% CI)")?;
    writeln!(s)?;
    writeln!(s, "| arm | Δ-Sharpe (vs baseline) | 95% CI |")?;
    writeln!(s, "|-----|-----------------------:|--------|")?;
    let baseline = result
        .arms
        .get(&cfg.baseline_arm)
        .ok_or_else(|| anyhow::anyhow!("baseline arm `{}` not in result", cfg.baseline_arm))?;
    for (name, arm) in &result.arms {
        if name == &cfg.baseline_arm { continue; }
        if arm.returns.len() != baseline.returns.len() {
            writeln!(s, "| `{}` | n/a (length mismatch) | — |", name)?;
            continue;
        }
        let r = bootstrap_delta_sharpe(
            &arm.returns,
            &baseline.returns,
            BootstrapConfig {
                n_resamples: cfg.n_bootstrap_resamples,
                block_size: cfg.block_size,
                seed: cfg.seed,
                periods_per_year: cfg.periods_per_year,
            },
        )?;
        writeln!(
            s,
            "| `{}` | {:+.4} | [{:+.4}, {:+.4}] |",
            name, r.point_estimate, r.ci_low, r.ci_high
        )?;
    }
    writeln!(s)?;

    // -- Per-arm dashboard --------------------------------------------
    writeln!(s, "## Per-arm dashboard")?;
    writeln!(s)?;
    writeln!(s, "| arm | n | Sharpe | MDD | PF | WR | realized $ |")?;
    writeln!(s, "|-----|--:|-------:|----:|---:|---:|-----------:|")?;
    for (name, arm) in &result.arms {
        let sh = sharpe_annualized(&arm.returns, 0.0, cfg.periods_per_year);
        let nav: Vec<f64> = arm.equity_curve.iter().map(|p| p.nav_usd).collect();
        let mdd = drawdown_max(&nav);
        let pf = profit_factor(&arm.returns);
        let wr = win_rate(&arm.returns);
        writeln!(
            s,
            "| `{}` | {} | {:.3} | {:.3} | {:.3} | {:.3} | {:.2} |",
            name, arm.returns.len(), sh, mdd, pf, wr, arm.realized_pnl_total_usd
        )?;
    }
    writeln!(s)?;

    // -- Regime stratification + gate ---------------------------------
    writeln!(s, "## Regime-stratified Δ-Sharpe + anti-overfit gate")?;
    writeln!(s)?;
    let pre = PreCommittedMetrics::from_result(result, &cfg.baseline_arm)?;
    for (treatment_name, by_regime) in &pre.delta_sharpe_by_regime {
        writeln!(s, "### Treatment `{treatment_name}`")?;
        writeln!(s, "| regime | Δ-Sharpe | 95% CI |")?;
        writeln!(s, "|--------|---------:|--------|")?;
        for (regime, br) in by_regime {
            writeln!(
                s,
                "| {regime:?} | {:+.4} | [{:+.4}, {:+.4}] |",
                br.point_estimate, br.ci_low, br.ci_high
            )?;
        }
        writeln!(s)?;
        let verdict = anti_overfit_gate(by_regime);
        match verdict {
            GateVerdict::PassesBothRegimes => writeln!(s, "**Gate: PassesBothRegimes**")?,
            GateVerdict::SingleRegimeEvidence { winning_regime, losing_regime } => writeln!(
                s,
                "**Gate: SingleRegimeEvidence — wins in {winning_regime:?}, loses in {losing_regime:?}**"
            )?,
            GateVerdict::Fails => writeln!(s, "**Gate: Fails**")?,
            other => writeln!(s, "**Gate: {other:?}**")?,
        }
        writeln!(s)?;
    }

    // -- Notes ---------------------------------------------------------
    writeln!(s, "## Notes")?;
    writeln!(s)?;
    writeln!(s, "- Δ-Sharpe is the only inferential metric. MDD, PF, WR are descriptive and not multiple-comparisons-corrected.")?;
    writeln!(s, "- Bootstrap: paired, n_resamples = {}, block_size = {:?}, seed = {}.", cfg.n_bootstrap_resamples, cfg.block_size, cfg.seed)?;
    writeln!(s, "- Anti-overfit gate is reportable but not blocking in v1 (see decisions/0005-lookahead-audit.md).")?;
    Ok(s)
}
```

```rust
/// One arm spec parsed from the CLI.
#[derive(Debug, Clone)]
pub struct ArmSpec {
    pub name: String,
    pub vector: VectorConfig,
}

/// Parse a CLI arm string. Forms accepted:
///   off
///   on:<npz_path>:<manifest_path>:<alpha>
///   random:layer=<u16>:dim=<usize>:alpha=<f32>:seed=<u64>
///   orthogonal:axis=<conviction|patience|risk|trend>:path=<npz>:alpha=<f32>:seed=<u64>
pub fn parse_arm_spec(s: &str) -> anyhow::Result<ArmSpec> {
    let mut parts = s.splitn(2, ':');
    let head = parts.next().unwrap_or("");
    let rest = parts.next().unwrap_or("");
    match head {
        "off" => Ok(ArmSpec { name: "vectors_off".into(), vector: VectorConfig::Off }),
        "on" => {
            let mut p = rest.splitn(3, ':');
            let npz = p.next().ok_or_else(|| anyhow!("on: missing npz path"))?;
            let manifest_path = p.next().ok_or_else(|| anyhow!("on: missing manifest path"))?;
            let alpha: f32 = p.next().unwrap_or("1.0").parse().context("on: alpha parse")?;
            let manifest_bytes = std::fs::read(manifest_path)?;
            let manifest: Manifest = serde_json::from_slice(&manifest_bytes)
                .context("on: parse manifest sidecar")?;
            Ok(ArmSpec {
                name: "vectors_on".into(),
                vector: VectorConfig::On {
                    manifest,
                    npz_path: PathBuf::from(npz),
                    alpha,
                },
            })
        }
        "random" => {
            let kv = parse_kv(rest);
            Ok(ArmSpec {
                name: "vectors_random".into(),
                vector: VectorConfig::Random {
                    seed: kv.get("seed").and_then(|s| s.parse().ok()).unwrap_or(42),
                    layer: kv.get("layer").and_then(|s| s.parse().ok()).unwrap_or(20),
                    hidden_dim: kv.get("dim").and_then(|s| s.parse().ok()).unwrap_or(5120),
                    alpha: kv.get("alpha").and_then(|s| s.parse().ok()).unwrap_or(1.0),
                },
            })
        }
        "orthogonal" => {
            let kv = parse_kv(rest);
            let axis = match kv.get("axis").map(String::as_str) {
                Some("conviction") | None => DispositionAxis::Conviction,
                Some("patience") => DispositionAxis::Patience,
                Some("risk") => DispositionAxis::Risk,
                Some("trend") => DispositionAxis::Trend,
                Some(other) => anyhow::bail!("orthogonal: unknown axis `{other}`"),
            };
            let path = kv
                .get("path")
                .map(PathBuf::from)
                .ok_or_else(|| anyhow!("orthogonal: missing path=<npz>"))?;
            Ok(ArmSpec {
                name: "vectors_orthogonal".into(),
                vector: VectorConfig::Orthogonal {
                    axis,
                    seed: kv.get("seed").and_then(|s| s.parse().ok()).unwrap_or(42),
                    npz_path: path,
                    alpha: kv.get("alpha").and_then(|s| s.parse().ok()).unwrap_or(1.0),
                },
            })
        }
        other => anyhow::bail!("unknown arm head: `{other}`"),
    }
}

fn parse_kv(s: &str) -> std::collections::BTreeMap<String, String> {
    s.split(':')
        .filter_map(|kv| {
            let mut it = kv.splitn(2, '=');
            let k = it.next()?.to_string();
            let v = it.next()?.to_string();
            Some((k, v))
        })
        .collect()
}

/// Run an N-arm A/B comparison. Returns the `BacktestResult` for serialization.
pub async fn run_ab_compare(
    snapshots: Vec<MarketSnapshot>,
    bars: Vec<MarketBar>,
    arms: Vec<ArmSpec>,
    config: BacktestRunConfig,
    intern: Arc<dyn InternBackend>,
    intern_provider: String,
    intern_model: String,
    engine: Arc<Mutex<Qwen3Engine>>,
    trader_params: TraderParams,
    portfolio_provider: Arc<dyn Fn() -> PortfolioState + Send + Sync>,
    risk: &RiskLayer,
) -> anyhow::Result<BacktestResult> {
    let cache = Arc::new(BriefingCache::new());

    let arm_configs: Vec<ArmConfig> = arms
        .into_iter()
        .map(|spec| {
            let arm: TraderArm = TraderArm::new(
                Box::leak(spec.name.clone().into_boxed_str()),
                Arc::clone(&intern),
                intern_provider.clone(),
                intern_model.clone(),
                Arc::clone(&cache),
                Arc::clone(&engine),
                trader_params.clone(),
                spec.vector,
                Arc::clone(&portfolio_provider),
            );
            ArmConfig {
                name: spec.name,
                strategy: Box::new(arm),
            }
        })
        .collect();

    let mut runner = BacktestRunner::new(config, arm_configs)?;
    let result = runner.run(&snapshots, &bars, risk).await?;
    Ok(result)
}
```

```rust
use std::path::PathBuf;
use std::sync::Arc;

use tokio::sync::Mutex;

use xianvec_core::market::MarketSnapshot;
use xianvec_core::trading::{AssetSymbol, PortfolioState};
use xianvec_eval::ab_compare::{parse_arm_spec, run_ab_compare};
use xianvec_eval::backtest::MarketBar;
use xianvec_eval::harness::BacktestRunConfig;
use xianvec_inference::engine::Qwen3Engine;
use xianvec_intern::backend::{AnthropicIntern, InternBackend, OpenAICompatIntern};
use xianvec_trader::TraderParams;

#[allow(clippy::too_many_arguments)]
pub async fn run(
    setups: PathBuf,
    bars: PathBuf,
    arms: String,
    output: PathBuf,
    initial_nav_usd: f64,
    fee_bps: u32,
    step_hours: u32,
    horizon_hours: u32,
    asset: String,
    model: PathBuf,
    tokenizer: PathBuf,
    intern_provider: String,
    intern_model: String,
) -> anyhow::Result<()> {
    let snapshots: Vec<MarketSnapshot> = serde_json::from_slice(&std::fs::read(&setups)?)?;
    let bars_vec: Vec<MarketBar> = serde_json::from_slice(&std::fs::read(&bars)?)?;
    let arm_specs: Vec<_> = arms
        .split(',')
        .map(|s| parse_arm_spec(s.trim()))
        .collect::<anyhow::Result<Vec<_>>>()?;

    let asset_sym = match asset.as_str() {
        "BTC" => AssetSymbol::Btc,
        "ETH" => AssetSymbol::Eth,
        "SOL" => AssetSymbol::Sol,
        other => anyhow::bail!("unknown asset: {other}"),
    };

    let engine = Qwen3Engine::load(&model, &tokenizer)?;
    let engine = Arc::new(Mutex::new(engine));

    let intern: Arc<dyn InternBackend> = match intern_provider.as_str() {
        "anthropic" => Arc::new(AnthropicIntern::from_env(
            "https://api.anthropic.com",
            &intern_model,
            "ANTHROPIC_API_KEY",
        )?),
        "openai-compat" => Arc::new(OpenAICompatIntern::from_env(
            std::env::var("OPENAI_BASE_URL").unwrap_or_else(|_| "https://api.openai.com/v1".into()),
            &intern_model,
            "OPENAI_API_KEY",
        )?),
        other => anyhow::bail!("unknown intern provider: {other}"),
    };

    let workspace_root = std::env::current_dir()?;
    let risk = xianvec_harness::load_risk_layer(&workspace_root)?;

    let cfg = BacktestRunConfig {
        initial_nav_usd,
        fee_bps,
        slippage_atr_frac: 0.0,
        instrument: asset_sym,
        step_hours,
        horizon_hours,
        n_bootstrap_resamples: 1000,
        block_size: None,
    };

    let portfolio_provider: Arc<dyn Fn() -> PortfolioState + Send + Sync> = Arc::new(|| {
        PortfolioState {
            equity_usd: 100_000.0,
            realized_pnl_today_usd: 0.0,
            day_index: 0,
            open_positions: Default::default(),
            as_of: chrono::Utc::now(),
        }
    });

    let result = run_ab_compare(
        snapshots,
        bars_vec,
        arm_specs,
        cfg,
        intern,
        intern_provider,
        intern_model,
        engine,
        TraderParams::default(),
        portfolio_provider,
        &risk,
    )
    .await?;

    std::fs::write(&output, serde_json::to_vec_pretty(&result)?)?;
    println!("wrote {} arm result(s) → {}", result.arms.len(), output.display());
    Ok(())
}
```
