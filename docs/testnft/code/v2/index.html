Xvision Bitfields v2

This pass removes the noisy overlays and follows the GitHub examples more closely: coarse grids, 5–8 opaque/transparent layers, row/radial banding, bitwise AND/XOR/OR, low state counts, strong palette discipline, and a bottom stamp.
