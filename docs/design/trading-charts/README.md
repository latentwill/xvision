## Key Findings

### Libraries & Tools
- **TradingView Lightweight Charts** — dominant open-source choice for financial charts (16K stars, v5.2.0 released Apr 2026)
- **Evilcharts** — shadcn/ui + recharts, handcrafted open-source chart UI (trending on X)
- **LightningChart JS** — GPU-rendered WebGL charts, 60 FPS, hundreds of millions of points
- **Recharts / D3 / Visx** — standard React charting stack
- **ECharts** — popular for admin dashboards with RSI, MACD, SMA/EMA, VWAP indicators

### Design Patterns
- Dark mode is the default expectation for trading UIs
- Green accent tones for positive/contrast are popular in crypto dashboards
- Data-dense layouts with drawers, tabs, and collapsible panels
- Custom toolbars, drawings, and indicators built on top of minimal chart libraries
- PNG export support for canvas charts is a common feature request

### Notable Projects
- **Kronos Studio** — added TradingView Lightweight Charts with candle/line toggle
- **ATLAS Terminal UI** — advanced charting module with full design system
- **MarketLens** — lightweight-charts framework with custom tools
- **MTF Candlestick Plugin** — multi-timeframe display without PineScript lock-in
