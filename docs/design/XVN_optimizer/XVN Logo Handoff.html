05 / Rules

Do · Don't

Hard rules — the bracket geometry is the brand
✓DO
Use the primary lockup on Carbon (#050505) for all in-product chrome.
Swap brackets to Signal Deep (#00B85F) on light backgrounds for legibility.
Use the all-green mono variant for stencils, stamps, and reversed-out chips.
Use the all-white variant when sitting next to vibrant chart colors that would clash with green.
✕DON'T
Don't rotate the lockup. The brackets are horizontal anchors.
Don't stretch or change the bracket-to-wordmark spacing.
Don't recolor the brackets outside the Signal family.
Don't swap typeface. Geist Mono 700 / 0.14em tracking is locked.
