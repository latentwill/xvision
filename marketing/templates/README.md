# marketing/templates/

Remotion video templates. Each template is a `.tsx` file that takes a
single data input (typically a `RunBundle` JSON) and renders a deterministic
mp4 via Remotion.

See `marketing/methods.md` § "Pillar 1" for the production pipeline and
the ranked build order.

## Adding a new template

1. **Input contract first.** Define the JSON shape the template consumes —
   typically a subset of an existing engine API response. Add an example
   fixture under `templates/__fixtures__/<template>.json`.
2. **One Remotion composition per template.** Don't share compositions across
   templates; copy is fine, coupling isn't.
3. **Reuse dashboard components.** TradingView Lightweight Charts, design
   tokens (`frontend/web/src/styles/tokens.css`), and the existing chart
   formatters all import into Remotion via standard ES modules.
4. **Audio is a separate step.** Templates render visuals; voiceover is
   generated by `scripts/generate-voiceover.sh` (TBD) calling 11labs and
   composited in via `<Audio src={...} />`.
5. **Reproducibility.** Each rendered mp4's metadata must include the
   `run_id` (or other deterministic input ID) it sourced from, so any viewer
   can re-run the same data and verify the video.

## Build commands (planned, not yet implemented)

```sh
# Generate voiceover from a script template
pnpm voice <template-name> --input fixtures/<run_id>.json

# Render a video locally
pnpm video <template-name> --input fixtures/<run_id>.json --out out/<run_id>.mp4

# Render on Remotion Lambda (when cadence > 4/wk)
pnpm video <template-name> --input fixtures/<run_id>.json --lambda
```

## Brand tokens

Pull from `frontend/web/src/styles/tokens.css`. Hard-coded color values are a
lint failure. Tailwind class names work if the Remotion project is configured
with the same Tailwind plugin as the dashboard.

| Token | Usage in video |
|---|---|
| `--bg` `#0F0E0C` | Frame background |
| `--gold` `#D4A547` | Accent — equity-curve highlight, key-moment markers |
| `--text` `#F1ECDD` | Primary text |
| `--text-3` | Secondary / labels |
| Cormorant Garamond italic | Title cards, episode names |
| Inter | All UI-like text |
| JetBrains Mono | Numbers (P&L, ratios, `run_id`s) |
