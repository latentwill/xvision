# xvision V2 board

> Fixture for migrate-board tests against the V2 board format.
> Uses em-dash + middle-dot row separators ("— · ·") to mirror the
> live team/board-v2.md format. The parser normalises these to ASCII
> hyphens before tokenising, so both forms produce identical output.
